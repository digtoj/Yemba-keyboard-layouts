Yemba Keyboard Layout

Type in Yemba language with ease across macOS and Windows platforms

Yemba: mɔ̀ · fɛ̀ · nə̀m · kɨ̂ · ɲàm · ŋwə̄

A comprehensive keyboard layout solution for the Yemba language (Grassfields Bantu, Cameroon)

🔤

Complete Character Set

All Yemba special characters including vowels with tone marks (ɛ̀, ɛ́, ɔ̀, ɔ́), central vowels (ə, ɨ, ʉ), and special consonants (ɲ, ŋ)

🖥️

Cross-Platform

Native support for macOS (QWERTY & QWERTZ) and Windows 11 (QWERTZ German) with easy installation scripts and graphical tools

⚡

Easy Installation

Multiple installation methods: automatic scripts, graphical tools (Ukulele for macOS), and manual installation with detailed guides

Platform Support

🍎

macOS

QWERTY (US)
Standard US keyboard layout
QWERTZ (German)
German keyboard with Y/Z swapped, umlauts
🚀 Automatic Script (Recommended)
cd macos/
./install-macos.sh
🎨 Ukulele (Graphical)
Visual keyboard editor with drag-and-drop installation

Quick Installation:

1
Download & Extract
Download the repository and navigate to the macOS folder
2
Run Installation Script
Execute ./install-macos.sh and follow the interactive menu
3
Activate Layout
Add the keyboard in System Preferences > Keyboard > Input Sources
🪟

Windows 11

QWERTZ (German)
Based on German layout with AltGr combinations
⚡ PowerShell Script
cd windows/qwertz-german
.\install-yemba-keyboard.ps1 -Install
🔧 Manual Installation
Using Microsoft Keyboard Layout Creator (MSKLC)

Quick Installation:

1
Install MSKLC
Download Microsoft Keyboard Layout Creator from Microsoft
2
Run PowerShell Script
Execute the installation script as Administrator
3
Add to Windows
Add keyboard in Settings > Language & Region > German > Keyboards

Troubleshooting

🍎

macOS Issues

Layout not appearing
Restart your Mac or log out and back in
Characters not displaying
Install Unicode fonts like Doulos SIL or Noto Sans
🪟

Windows Issues

Installation fails
Run PowerShell as Administrator and ensure MSKLC is installed
AltGr not working
Use the right Alt key (AltGr) and ensure German language pack is installed
